Room Allocation Settings

 The Room Allocation process allows students to view rooms in a table along with their capacities. Students can apply for a room with a pre-formed group if the group's size matches the room's capacity. This process should be started only after all rooms have been added by the admin and should allow some days for students to form groups. After starting, it should end after a designated period, at which point an algorithm will automatically place remaining students into rooms. Start Room Allocation Process End Room Allocation Process
